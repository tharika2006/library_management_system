# Library Management System

## Project Overview

The **Library Management System** is a web-based application designed to manage and organize library operations efficiently. The system allows users to view books, manage members, issue books, and access library information through a simple and user-friendly interface.

This project demonstrates how basic **web technologies such as HTML, CSS, and simple web structure** can be used to build an organized library management interface.

## Objectives

* To simplify the process of managing library records.
* To provide an organized interface for book and member management.
* To demonstrate basic web application development.
* To help understand how real-world systems like library management are structured.

## Features

* 📚 View available books in the library
* 👥 Manage library members
* 📖 Issue books to members
* 🏠 Library home page with navigation
* 📞 Contact page for library information
* Easy navigation between pages

## Project Structure

The project contains multiple HTML pages representing different modules of the system.

```
Library-Management-System
│
├── lib-home.html          # Home page of the system
├── lib-books.html         # Displays available books
├── lib-members.html       # Shows library members
├── lib-issuebooks.html    # Book issuing interface
├── lib-contact.html       # Contact page
└── .vscode                # VS Code configuration files
```

## Technologies Used

* **HTML5** – For structuring web pages
* **CSS** – For styling and layout
* **VS Code** – Development environment

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **lib-home.html** in any web browser.
4. Navigate through the system using the menu options.
